### 简介
这个基于beego框架的app,旨在熟悉beego框架下mvc的使用，以及go语言几web知识的学习


### js
- npm init
- webpack
```
npm install --save-dev webpack
```
- 打包编译
```
node_modules/.bin/webpack
```

### Webpack的强大功能
- 生成Source Maps（使调试更容易）
- 使用webpack构建本地服务器
```
npm install --save-dev webpack-dev-server
```
- Loaders

### vue
- install
```
npm install --save-dev vue
```
